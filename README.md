IT Ticket Data Analysis Portfolio
An interactive data analysis project exploring 97,498 IT support tickets spanning from 2016 to 2020. This project demonstrates proficiency in data cleaning, exploratory data analysis (EDA), and interactive visualization.

Live Demo: adiexel.netlify.app

📊 Project Overview
This analysis uncovers actionable insights from a high-volume IT Helpdesk dataset. The goal was to identify growth trends, evaluate agent performance, and understand resolution patterns across different technical categories.

Key Metrics
Total Tickets: 97,498

Data Range: 5 Years (2016–2020)

Avg Satisfaction: 4.10 / 5.0

Avg Resolution Time: 4.55 Days

🛠️ Tech Stack & Tools
Data Processing: Microsoft Excel (Power Query, Pivot Tables)

Frontend/UI: HTML5, CSS3, JavaScript (ES6+)

Data Visualization: Chart.js

Deployment: Netlify

📈 Key Insights
Growth Trend: Observed a 123% increase in ticket volume from 2016 to 2020, signaling rapid business expansion.

Resolution Efficiency: 'Login Access' issues are resolved in just 0.31 days on average, significantly faster than 'Hardware' issues (7.63 days).

Severity Distribution: 90.9% of tickets are classified as 'Normal' severity, indicating a stable IT environment despite high volume.

Consistency: Customer satisfaction remained remarkably stable (4.09–4.11) across all five years.

🚀 Features
Interactive Dashboard: Real-time KPI tracking.

Dynamic Explorer: Filter nearly 100k rows of data by Year, Category, Severity, and Priority.

Agent Leaderboard: Performance tracking for 50 IT agents based on volume and satisfaction scores.

Responsive Design: Fully optimized for desktop and mobile viewing.

📁 Dataset Structure
The analysis was performed on two primary datasets:

Tickets Sheet: 97,498 rows × 13 columns (ID, Date, Category, Severity, Priority, etc.)

IT Agents Sheet: 50 records × 7 columns (Agent Name, Department, etc.)
