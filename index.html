05 About This Project

📊 Dataset Overview

This project analyzes an IT Helpdesk ticketing system dataset containing 97,498 real-world support tickets from 2016 to 2020. The data covers four request categories — System, Login Access, Software, and Hardware — across 50 dedicated IT agents.

The dataset was structured with two primary sheets: the Tickets sheet (97,498 rows × 13 columns) and the IT Agents sheet (50 records × 7 columns), providing a comprehensive view of IT support operations over five years.

🎯 Project Objectives

The analysis aims to uncover actionable insights: identifying peak ticket periods, evaluating agent performance, understanding resolution time patterns, and measuring customer satisfaction across categories and severity levels.

Key findings include a 123% growth in ticket volume from 2016 to 2020, Login Access tickets resolving 20x faster than Hardware tickets, and remarkably consistent satisfaction scores (4.09–4.11) across all categories.

🛠️ Tools & Stack

This portfolio project was built as an interactive Excel data analysis project, demonstrating proficiency in data cleaning, pivot analysis, dashboard creation, and now web-based visualization.

💡 Key Insights

+123% ticket growth from 2016→2020, signaling rapid business/IT expansion.

Login Access resolves in just 0.31 days on average — far ahead of all other categories.

Hardware takes 7.63 days — nearly 25x longer than Login Access tickets.

90.9% of tickets are classified as Normal severity, indicating stable IT operations.
